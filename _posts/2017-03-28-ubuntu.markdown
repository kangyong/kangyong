#### 3T+ disk mount
分区转换成GPT即可
sudo parted /dev/sdb
将MBR硬盘格式化为GPT
mklabel gpt
之后可以看一下状态
print
整个硬盘空间只分一个区
mkpart primary 0 -1
再看一下
print
没问题，退出
quit

sudo mkfs -t ext4 /dev/sdb1
sudo mount -t ext4 /dev/sdb1 /mapbar/
